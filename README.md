# Stock-Management-system-vb.net
Stock management and ordering system built with VB.NET and MS Access.
## Features
- Add, update, delete, and search products
- Manage supplier information
- Place and track orders
- Database integration using MS Access
- Input validation and basic defensive programming

  # Technologies Used
- VB.NET (Visual Studio)
- MS Access (Database)

 # Purpose
This project was developed as part of my academic coursework to demonstrate:
- Object-Oriented Programming
- Database connectivity
- Business logic implementation
- User-friendly interface design
